API

Magic Console is composed of payloads, runtimes, and plugins, each of which have their own API for extension and customization.

  • Payloads: Payload refers to the output data that is sent from the source code to the output panel.
  • Plugins: Plugin refers to React components that are used to render output data into rich visualizations.
  • Runtimes: Runtime refers to configuration mapping Atom grammars to runtime commands.
